# Online Auction System (Terminal-Based)

## Description
This is a terminal-based Online Auction System developed using C.
The project allows users to list items, place bids, and determine
the highest bidder using a menu-driven console interface.

## Features
- User registration
- Add items for auction
- Place bids
- Display highest bid
- Menu-driven terminal interface

## Technologies Used
- Programming Language: C
- Concepts Used:
  - Structures
  - Functions
  - File Handling
  - Control Statements

## How to Run
1. Compile the program:
   gcc auction.c -o auction
2. Run the executable:
   ./auction
